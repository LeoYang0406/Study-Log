# Study-Log

1. .equals() 和 == 的使用情景
  基本数据类型（byte, short, int, long, double, float, boolean, char）
      
  引用数据类型（class, Interface, )
      变量名指向此object的地址
      
2. .equals() 和 == 的使用情景
    当数据类型为基本类型时， 可以用==
    当数据类型为引用数据类型时，用.equals()
    
3. .equals()的原理
    将此变量下的所有data和另一比较对象的所有data一一比较
